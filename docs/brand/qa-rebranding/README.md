# QA visual del rebranding (KER-24)

Capturas del cierre de la época de rebranding (KER-19…KER-24): las mismas nueve
pantallas, recorridas con los tres roles (familia, cuidador, admin), en la
identidad v1 y en la identidad v2 "abrazo profesional" del
[brand book](../brand-book.md).

- **`antes/`** — Keru-Webapp en `266fd16` (último commit de `main` previo al
  merge de KER-20, el primer PR del design system v2).
- **`despues/`** — Keru-Webapp con la época completa (KER-20…KER-24).

Tomadas contra el entorno local con seed, viewport 1280×832, `prefers-reduced-motion`.
